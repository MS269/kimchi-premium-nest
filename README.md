# Kimchi Premium Nest

NestJS + TypeORM + Oracle/SQLite

## Description

Kimchi Premium NestJS Server

## Installation

Install Node.js 20.x.x LTS

```bash
$ npm install
```

```bash
$ cp .env.example .env
```

Setup .env file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
